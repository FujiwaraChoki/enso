# Security Policy

## Supported Versions

We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security vulnerability, please follow these steps:

1. **Do not** open a public GitHub issue
2. Email the details to [sami@samihindi.com] with the subject line "Security Vulnerability: [Brief Description]"
3. Include the following information:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if you have one)

### What to Expect

- You will receive an acknowledgment within 48 hours
- We will investigate and provide an initial assessment within 7 days
- We will keep you informed of our progress
- Once the vulnerability is resolved, we will credit you (if desired) in our security advisories

### Scope

Please report security vulnerabilities in:
- The Enso application codebase
- Dependencies that affect Enso's security
- Configuration issues that could compromise user data

### Out of Scope

The following are considered out of scope:
- Social engineering attacks
- Physical attacks
- Denial of service attacks
- Issues requiring physical access to a user's device
- Issues in third-party services or dependencies that are not directly maintained by this project

## Security Best Practices

When using Enso:

- Keep your macOS system updated
- Use strong, unique passwords for your email accounts
- Enable two-factor authentication on your email accounts when available
- Review the permissions requested by the application
- Report any suspicious behavior immediately

## Disclosure Policy

- We will publicly disclose vulnerabilities after they have been fixed and a patch is available
- We will credit security researchers who responsibly report vulnerabilities (with their permission)
- We aim to provide security updates in a timely manner

Thank you for helping keep Enso and its users safe!
