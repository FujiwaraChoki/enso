## [1.0.0] - 2025-12-19

### Added
- Multi-account IMAP/SMTP email support
- On-device AI assistant powered by Apple's Foundation Models
- Liquid Glass design system with Instrument Serif typography
- Browser-style tab interface for email management
- SwiftData persistence for local data storage
- Secure credential storage using macOS Keychain
- Real-time email synchronization with IMAP IDLE
- Email composition and sending
- Attachment support
- Search functionality
- Onboarding flow for new users

### Security
- All credentials stored securely in macOS Keychain
- Network communications use TLS encryption
- No sensitive data stored in plain text
